Dicey Dungeons Switch Animation Speed Bug

Hello! Unfortunately, there is a bug with the "Animation Speed" setting in the Nintendo Switch version of Dicey Dungeons. The bug is that if you decrease this setting, you are unable to increase it again. We will fix this in an upcoming patch, but here is a workaround in the meantime:

The workaround is:

  • Highlight the BACK button using the Left Stick.
  • Use the touchscreen to change the value in the slider.
  • While keeping your finger held on the screen, press UP on the Left Stick to select the animation speed slider.

The slider should now stay in place.
